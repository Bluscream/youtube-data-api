This is the API

youtube.search.list

Retrieves a list of search resources

URL

https://youtube.googleapis.com//youtube/v3/search

HTTP Method

GET

Response Type

SearchListResponse

Parameters

  • Query: part List(String)

    The *part* parameter specifies a comma-separated list of one or more search resource properties that the API response will include. Set the parameter value to snippet.

  • Query: $.xgafv String

    V1 error format.

  • Query: access_token String

    OAuth access token.

  • Query: alt String

    Data format for response.

  • Query: callback String

    JSONP

  • Query: fields String

    Selector specifying which fields to include in a partial response.

  • Query: key String

    API key. Your API key identifies your project and provides you with API access, quota, and reports. Required unless you provide an OAuth 2.0 token.

  • Query: oauth_token String

    OAuth 2.0 token for the current user.

  • Query: prettyPrint Boolean

    Returns response with indentations and line breaks.

  • Query: quotaUser String

    Available to use for quota purposes for server-side applications. Can be any arbitrary string assigned to a user, but should not exceed 40 characters.

  • Query: upload_protocol String

    Upload protocol for media (e.g. \"raw\", \"multipart\").

  • Query: uploadType String

    Legacy upload protocol for media (e.g. \"media\", \"multipart\").

  • Query: channelId String

    Filter on resources belonging to this channelId.

  • Query: channelType String

    Add a filter on the channel search.

  • Query: eventType String

    Filter on the livestream status of the videos.

  • Query: forContentOwner Boolean

    Search owned by a content owner.

  • Query: forDeveloper Boolean

    Restrict the search to only retrieve videos uploaded using the project id of the authenticated user.

  • Query: forMine Boolean

    Search for the private videos of the authenticated user.

  • Query: location String

    Filter on location of the video

  • Query: locationRadius String

    Filter on distance from the location (specified above).

  • Query: maxResults Integer

    The *maxResults* parameter specifies the maximum number of items that should be returned in the result set.

  • Query: onBehalfOfContentOwner String

    *Note:* This parameter is intended exclusively for YouTube content partners. The *onBehalfOfContentOwner* parameter indicates that the request's authorization credentials identify a YouTube CMS user who is acting on behalf of the content owner specified in the parameter value. This parameter is intended for YouTube content partners that own and manage many different YouTube channels. It allows content owners to authenticate once and get access to all their video and channel data, without having to provide authentication credentials for each individual channel. The CMS account that the user authenticates with must be linked to the specified YouTube content owner.

  • Query: order String

    Sort order of the results.

  • Query: pageToken String

    The *pageToken* parameter identifies a specific page in the result set that should be returned. In an API response, the nextPageToken and prevPageToken properties identify other pages that could be retrieved.

  • Query: publishedAfter String

    Filter on resources published after this date.

  • Query: publishedBefore String

    Filter on resources published before this date.

  • Query: q String

    Textual search terms to match.

  • Query: regionCode String

    Display the content as seen by viewers in this country.

  • Query: relatedToVideoId String

    Search related to a resource.

  • Query: relevanceLanguage String

    Return results relevant to this language.

  • Query: safeSearch String

    Indicates whether the search results should include restricted content as well as standard content.

  • Query: topicId String

    Restrict results to a particular topic.

  • Query: type List(String)

    Restrict results to a particular set of resource types from One Platform.

  • Query: videoCaption String

    Filter on the presence of captions on the videos.

  • Query: videoCategoryId String

    Filter on videos in a specific category.

  • Query: videoDefinition String

    Filter on the definition of the videos.

  • Query: videoDimension String

    Filter on 3d videos.

  • Query: videoDuration String

    Filter on the duration of the videos.

  • Query: videoEmbeddable String

    Filter on embeddable videos.

  • Query: videoLicense String

    Filter on the license of the videos.

  • Query: videoSyndicated String

    Filter on syndicated videos.

  • Query: videoType String

    Filter on videos of a specific type.
