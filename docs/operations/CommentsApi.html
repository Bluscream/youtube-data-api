youtube.comments.list

Retrieves a list of resources, possibly filtered.

https://youtube.googleapis.com//youtube/v3/comments

HTTP Method

GET

Response Type

CommentListResponse

Parameters

  • Query: part List(String)

    The *part* parameter specifies a comma-separated list of one or more comment resource properties that the API response will include.

  • Query: $.xgafv String

    V1 error format.

  • Query: access_token String

    OAuth access token.

  • Query: alt String

    Data format for response.

  • Query: callback String

    JSONP

  • Query: fields String

    Selector specifying which fields to include in a partial response.

  • Query: key String

    API key. Your API key identifies your project and provides you with API access, quota, and reports. Required unless you provide an OAuth 2.0 token.

  • Query: oauth_token String

    OAuth 2.0 token for the current user.

  • Query: prettyPrint Boolean

    Returns response with indentations and line breaks.

  • Query: quotaUser String

    Available to use for quota purposes for server-side applications. Can be any arbitrary string assigned to a user, but should not exceed 40 characters.

  • Query: upload_protocol String

    Upload protocol for media (e.g. \"raw\", \"multipart\").

  • Query: uploadType String

    Legacy upload protocol for media (e.g. \"media\", \"multipart\").

  • Query: id List(String)

    Returns the comments with the given IDs for One Platform.

  • Query: maxResults Integer

    The *maxResults* parameter specifies the maximum number of items that should be returned in the result set.

  • Query: pageToken String

    The *pageToken* parameter identifies a specific page in the result set that should be returned. In an API response, the nextPageToken and prevPageToken properties identify other pages that could be retrieved.

  • Query: parentId String

    Returns replies to the specified comment. Note, currently YouTube features only one level of replies (ie replies to top level comments). However replies to replies may be supported in the future.

  • Query: textFormat String

    The requested text format for the returned comments.

youtube.comments.setModerationStatus

Sets the moderation status of one or more comments.

URL

https://youtube.googleapis.com//youtube/v3/comments/setModerationStatus

HTTP Method

POST

Response Type

Parameters

  • Query: id List(String)

    Modifies the moderation status of the comments with the given IDs

  • Query: moderationStatus String

    Specifies the requested moderation status. Note, comments can be in statuses, which are not available through this call. For example, this call does not allow to mark a comment as 'likely spam'. Valid values: MODERATION_STATUS_PUBLISHED, MODERATION_STATUS_HELD_FOR_REVIEW, MODERATION_STATUS_REJECTED.

  • Query: $.xgafv String

    V1 error format.

  • Query: access_token String

    OAuth access token.

  • Query: alt String

    Data format for response.

  • Query: callback String

    JSONP

  • Query: fields String

    Selector specifying which fields to include in a partial response.

  • Query: key String

    API key. Your API key identifies your project and provides you with API access, quota, and reports. Required unless you provide an OAuth 2.0 token.

  • Query: oauth_token String

    OAuth 2.0 token for the current user.

  • Query: prettyPrint Boolean

    Returns response with indentations and line breaks.

  • Query: quotaUser String

    Available to use for quota purposes for server-side applications. Can be any arbitrary string assigned to a user, but should not exceed 40 characters.

  • Query: upload_protocol String

    Upload protocol for media (e.g. \"raw\", \"multipart\").

  • Query: uploadType String

    Legacy upload protocol for media (e.g. \"media\", \"multipart\").

  • Query: banAuthor Boolean

    If set to true the author of the comment gets added to the ban list. This means all future comments of the author will autmomatically be rejected. Only valid in combination with STATUS_REJECTED.
