youtube.commentThreads.insert

Inserts a new resource into this collection.

URL

https://youtube.googleapis.com//youtube/v3/commentThreads

HTTP Method

POST

Response Type

CommentThread

Parameters

  • Query: part List(String)

    The *part* parameter identifies the properties that the API response will include. Set the parameter value to snippet. The snippet part has a quota cost of 2 units.

  • Body: body CommentThread(CommentThread)

  • Query: $.xgafv String

    V1 error format.

  • Query: access_token String

    OAuth access token.

  • Query: alt String

    Data format for response.

  • Query: callback String

    JSONP

  • Query: fields String

    Selector specifying which fields to include in a partial response.

  • Query: key String

    API key. Your API key identifies your project and provides you with API access, quota, and reports. Required unless you provide an OAuth 2.0 token.

  • Query: oauth_token String

    OAuth 2.0 token for the current user.

  • Query: prettyPrint Boolean

    Returns response with indentations and line breaks.

  • Query: quotaUser String

    Available to use for quota purposes for server-side applications. Can be any arbitrary string assigned to a user, but should not exceed 40 characters.

  • Query: upload_protocol String

    Upload protocol for media (e.g. \"raw\", \"multipart\").

  • Query: uploadType String

    Legacy upload protocol for media (e.g. \"media\", \"multipart\").

youtube.commentThreads.list

Retrieves a list of resources, possibly filtered.

URL

https://youtube.googleapis.com//youtube/v3/commentThreads

HTTP Method

GET

Response Type

CommentThreadListResponse

Parameters

  • Query: part List(String)

    The *part* parameter specifies a comma-separated list of one or more commentThread resource properties that the API response will include.

  • Query: $.xgafv String

    V1 error format.

  • Query: access_token String

    OAuth access token.

  • Query: alt String

    Data format for response.

  • Query: callback String

    JSONP

  • Query: fields String

    Selector specifying which fields to include in a partial response.

  • Query: key String

    API key. Your API key identifies your project and provides you with API access, quota, and reports. Required unless you provide an OAuth 2.0 token.

  • Query: oauth_token String

    OAuth 2.0 token for the current user.

  • Query: prettyPrint Boolean

    Returns response with indentations and line breaks.

  • Query: quotaUser String

    Available to use for quota purposes for server-side applications. Can be any arbitrary string assigned to a user, but should not exceed 40 characters.

  • Query: upload_protocol String

    Upload protocol for media (e.g. \"raw\", \"multipart\").

  • Query: uploadType String

    Legacy upload protocol for media (e.g. \"media\", \"multipart\").

  • Query: allThreadsRelatedToChannelId String

    Returns the comment threads of all videos of the channel and the channel comments as well.

  • Query: channelId String

    Returns the comment threads for all the channel comments (ie does not include comments left on videos).

  • Query: id List(String)

    Returns the comment threads with the given IDs for Stubby or Apiary.

  • Query: maxResults Integer

    The *maxResults* parameter specifies the maximum number of items that should be returned in the result set.

  • Query: moderationStatus String

    Limits the returned comment threads to those with the specified moderation status. Not compatible with the 'id' filter. Valid values: published, heldForReview, likelySpam.

  • Query: order String

  • Query: pageToken String

    The *pageToken* parameter identifies a specific page in the result set that should be returned. In an API response, the nextPageToken and prevPageToken properties identify other pages that could be retrieved.

  • Query: searchTerms String

    Limits the returned comment threads to those matching the specified key words. Not compatible with the 'id' filter.

  • Query: textFormat String

    The requested text format for the returned comments.

  • Query: videoId String

    Returns the comment threads of the specified video.
